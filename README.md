### Installation

#### Clone repo

```bash
# Clone the repo
$ git clone https://github.com/Agusdeluca96/agileengine-image-gallery.git

# Go into app's directory
$ cd agileengine-image-gallery

# Install app's dependencies
$ npm install
```

#### Usage

1. Start de server

   ```bash
   npm run serve
   ```

2. Open project on browser: http://localhost:8080/
